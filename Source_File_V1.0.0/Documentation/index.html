3. Assets Overview


All asset-related files for this template are organized in the assets folder. The folder structure is designed to make customization straightforward and efficient. Below is the updated folder map for your gallery template:

assets/
|-- bootstrap/                  # Contains Bootstrap framework files
|-- css/                        # Contains all stylesheet files
|-- fonts/                      # Contains font files
|-- image/                      # Contains all template images
|-- js/                         # Contains JavaScript files

The css folder contains individual CSS files for each gallery template, allowing you to easily modify the styles for specific pages.

When customizing, avoid modifying the HTML code directly. Instead, make changes to the relevant CSS file. The CSS files include media queries to ensure responsiveness across all device sizes. Familiarizing yourself with these files will help you achieve the desired design and layout for your gallery templates.

4. Support

Thank you for choosing our gallery templates! We are committed to assisting you with any questions or issues you may encounter. Our support covers answering queries and resolving bugs related to the templates.

Please note that our support does not include customization or installation services, nor assistance with third-party tools or plugins. However, we will do our best to provide guidance wherever possible.
